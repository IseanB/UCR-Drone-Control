# UCR Drone Control

## Installation 
  Tip: Only use **catkin build**, never use **catkin_make**.
  1. Install PX4 Related Dependencies. [^1]
 
  2. Install ROS Melodic. [^2]

  3. Install Simulation Common Dependencies. [^3]

  4. Install MAVROS & Setup Workspace [^4]
